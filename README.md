One Two Three!
====

## Description

世界のナベアツさんに芸が好きだったのでそれをゲームにしてみました！

## ルール

1~100のうち、「3の倍数と3のつく数」をその他から分類していきます。
連続で間違えずに分類できた数とそのタイムでスコアが決まります。
